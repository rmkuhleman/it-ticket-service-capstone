# IT Ticket Service Capstone

This capstone project focuses on creating an IT ticket service that allows users to submit technical issues, enables technicians to review and update tickets, and supports escalation, resolution, and feedback.

## Current Status
The project has progressed from planning into a functional implementation phase. Core form functionality, backend validation, and structured error handling have been implemented. The system now supports full ticket submission with improved user feedback and data validation.\
