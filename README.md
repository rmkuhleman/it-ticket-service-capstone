# IT Ticket Service Capstone

This capstone project focuses on creating an IT ticket service that allows users to submit technical issues, enables technicians to review and update tickets, and supports escalation, resolution, and feedback.

## Current Status
The project is currently in the frontend development phase. The user interface for the IT ticket submission system has been built, including the form layout, input fields, category and subcategory selection behavior, and a confirmation page for submitted data. At this stage, the application is a frontend prototype and does not yet include backend processing, data storage, or server-side validation. The next phase will focus on integrating backend functionality using Flask to handle form submissions and implement validation logic.
